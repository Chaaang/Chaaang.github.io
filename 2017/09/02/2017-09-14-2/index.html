开源项目 Morning

发表于 2017-09-02 | 分类于 开源项目 | 阅读次数
字数统计 480
  • 项目地址

面向用户

有一定动手能力的普通用户和程序员

使用场景

是打开terminal时自动对某些网站监测并进行规律化操作(如 签到，打卡，发早安，顶帖，内容筛选等)

支持平台

主要是 MacOS和Linux (手里没Win的笔记本)

使用方法

1. 克隆项目

git clone https://github.com/Chaaang/morning.git&& cd morning

2. 安装依赖项

pip install -r requirements.txt

3. 配置

使用 chrome 浏览器安装 EditThisCookie 插件，将页面cookies导出到剪贴板并放在cookies文件夹下以站点域名打头的txt文件里(如jobbole.com.txt) 推荐站点(v2ex, 网易云音乐，jobbole)。
如果不采取这种方式保存的话将会自动读取firefox的cookies，如果读取失败会报错。

如果不想使用某些网站的话，请修改config.py 内 list_black 数组

技巧 1 手动调用

alias morning="python3 /Users/c/PycharmProjects/morning/morning.py"

最近更新

  • 将操作结果保存并对未成功的操作可以再次尝试

登录逻辑

登陆部分选择省去，由用户将浏览器的cookies提取出来。目前的可选项是:

  • Chrome浏览器: 使用 EditThisCookie 插件
  • 自动从 Firefox 浏览器内获取Cookies (系统默认路径下)
    • 网易音乐无法使用这种方式登陆
